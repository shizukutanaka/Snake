# Snake

GPUスケジューラ + Windows最適化ユーティリティ

## 特徴

- CUDA/非CUDA混在GPU対応（NVIDIA, AMD, Intel）
- スケジューリング戦略（Static, Dynamic, Adaptive）
- 起動高速化・ネット最適化（Windows向け）
- プロファイリング出力（CSV/JSON）
- 軽量常駐（従来の1%未満のリソース使用）
- C++17 / CMake対応


## ビルド方法（例：Linux）

```bash
mkdir build
cd build
cmake ..
make
```
